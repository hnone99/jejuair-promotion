Button Style Guide

[2021-05-11]

Action button - primary : 페이지 이동버튼(프로세스 진행시 페이지 이동버튼으로 주로 쓰이며 화면 하단에 fixed로 위치)
[PC]

- height:50px; min-width:110px, 버튼 내 좌우 padding:40px; font-size:16px; font-weight:700
- 페이지 이동버튼 (프로세스 진행 상 꼭 필요한 버튼으로 페이지 상 하나만 존재)
- disable시 'button--active' class 추가
- Normal-Black : 버튼 두개가 나란히 올경우 부정케이스 적용 (아니오, 취소)

[Mobile]

- height:56px; font-size:16px; font-weight:700
- 페이지 이동버튼 (프로세스 진행 상 꼭 필요한 버튼으로 페이지 상 하나만 존재)
- disable시 'button--active' class 추가
- Normal-Black : 버튼 두개가 나란히 올경우 부정케이스 적용 (아니오, 취소)
- Nomal-Action과 Floating-Action 두가지 스타일로 나뉨
＊Nomal-Action : 회원가입, 로그인과 같은 프로세스가 있는 페이지경우,
   페이지 중간에 올 경우(확실치 않음)
＊Floating-Action :유플리트 기획에 fixed라고 정의 된 버튼은 Floating-Action
   스타일이라고 확인함(대부분페이지)
