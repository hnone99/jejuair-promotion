  • data-element="tab" : 최상위 부모 dom 속성
  • class="tab__button-wrap" : 스크롤을 담당하는 돔요소
  • data-element="tab__list" : 각 요소 클릭시 좌측 끝으로 스크롤
  • 값 모두 입력시 is-checked 클래스 추가

Event

  • beforeChange: 패널 오픈 전 실행될 함수(Arguments: event, plugin, anchor, panel)
  • afterChange: 패널 오픈 후 실행될 함수(Arguments: event, plugin, anchor, panel)

Method

  • open: 원하는 탭 오픈(Arguments: index, autoScroll(Boolean): 선택된 탭의 상단으로 스크롤)
  • destroy: 플러그인 제거(Arguments: none)
  • reInit: 플러그인 제거 후 생성(Arguments: none)
