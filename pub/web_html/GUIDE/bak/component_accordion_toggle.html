data-element="toggle"

  • 항공권 구매 시, 회원번호를 입력하지 않고 구매하고 탑승한 내역은 조회하실 수 없습니다.
  • 제주항공 홈페이지에서 예매한 탑승 내역에 대한 결제 정보만 조회하실 수 있습니다.
  • 회원가입 이후의 탑승 내역만 조회하실 수 있습니다.
  • 최근 3년 이내 본인이 탑승했던 내역만 조회하실 수 있습니다.
  • ‘적립 예정‘은 항공권 구매 후 적립에 필요한 기간 내에서 처리 중임을 뜻하며, 처리 완료 후 포인트 및 탑승 횟수가 적립됩니다. 
        <div class="service-information" data-element="toggle" data-options='{"mode": "move scroll"}'>
          <div class="service-information__title">
            <button type="button" class="service-information__anchor" data-element="toggle__anchor">탑승 내역 조회 안내</button>
          </div>
          <div class="service-information__content" data-element="toggle__panel">
            ...
          </div>
        </div>
  • data-element="toggle" : accordion과 동일한 UI로 toggle 형태로 적용 가능
  • data-element="toggle__panel" : 타이틀 클릭시 display 됨
  • data-options='{"mode": "slide"}' : 슬라이드 transition 부드럽게 처리
  • data-options='{"mode": "move scroll"}' : PC는 단순 오픈 / MOBILE 아래에서 위로 올림
  • 포인트 추가 적립
  • 위탁 수하물 20kg
  • 좌석 선택 일반석 무료
        <div class="bundle_box" data-element="toggle" data-options='{"mode": "slide"}'">
          <div class="bundle_box__btn_wrap">
            ...
          </div>
          <div class="bundle_box__content" data-element="toggle__panel">
            ...
          </div>
        </div>

data option 목록

        data-element="accordion" data-options='{ "autoFold": false }'
  • autoScroll : 화면 스크롤 이동(true / false)
  • autoFold : 자동 닫힘(true / false)
  • initIndex : 초기화시 열릴 패널의 index(0)

Event

        var accordionItem = $('.your-element');
        accordionItem.on('beforeChange', function(event, plugin, anchor) {
          /* your code ... */
        })
  • beforeChange : 패널 오픈 전 실행될 함수(arguments: event, plugin, anchor)
  • afterChange : 패널 오픈 후 실행될 함수(arguments: event, plugin, anchor)
