PC 웹뷰은 1004px에 맞춰져있고, 1080px부터 모바일웹뷰 (브레이크포인트)
  • global.css - 공통정의, 폰트스타일 지정
  • component.css - 컴포넌트별 스타일(PC, 모바일, 320)
  • page.css - 모바일웹을 기본으로 CSS정의가 됨
  • responsive.css - 반응형이 적용된 PC화면
  • main.css - 메인만 쓰는 CSS, page.css에 import
  • ticketing.css - 퀵부킹 공통영역 CSS, page.css에 import
  • jjpay.css - 모달 새창, 별도로 관리해야함 (모바일, PC 공통 한스타일)
  • dark_main.css - 다크메인만 쓰는 CSS
  • lang.css - 언어별 CSS 다국어)
  • front_global.css - CMS 미리보기영역
  • front_main.css - CMS 미리보기영역 (main.css 카피해서 개발전달)
  • front_page.css - CMS 미리보기영역 (page.css 카피해서 개발전달)
